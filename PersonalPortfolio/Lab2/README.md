# 🔀 Lab 2 — Network Expansion & VLANs

## 📋 Overview

Second lab expanding on the basic LAN by introducing VLANs (Virtual Local Area Networks), trunking, and inter-VLAN routing concepts.

## 📄 Files

| File | Description |
|------|-------------|
| `Lab2 Network.docx` | Lab report and guide (Word) |
| `Lab2 Network.pdf` | Lab report and guide (PDF) |
| `lab2.pkt` | Cisco Packet Tracer simulation |

## 🎯 Learning Objectives

- Configure VLANs on a managed switch
- Set up trunk links between switches
- Implement inter-VLAN routing via a Layer 3 device
- Test isolation between VLANs and routing between them

## 🛠️ Technologies

- Cisco Packet Tracer
- Layer 2 Managed Switches
- 802.1Q VLAN Trunking
- Router-on-a-Stick / Layer 3 switch routing

## 🚀 Getting Started

1. Open `lab2.pkt` in Cisco Packet Tracer
2. Follow the guide in the PDF/DOCX
3. Verify VLAN isolation and inter-VLAN connectivity
