# 🖥️ Lab 1 — Basic LAN Setup

## 📋 Overview

First lab introducing fundamental LAN configuration — connecting devices, assigning IP addresses, and verifying basic network connectivity using Cisco Packet Tracer.

## 📄 Files

| File | Description |
|------|-------------|
| `lab 1 Complete Lab-Focused Teaching Package.docx` | Lab guide (Word document) |
| `lab 1 Complete Lab-Focused Teaching Package.pdf` | Lab guide (PDF) |
| `lab1.pkt` | Cisco Packet Tracer simulation |

## 🎯 Learning Objectives

- Build a basic Local Area Network (LAN)
- Configure static IP addresses on end devices
- Verify connectivity using `ping` and `ipconfig`
- Understand Layer 2 switching behavior

## 🛠️ Technologies

- Cisco Packet Tracer
- Switches & PCs
- Ethernet cabling (straight-through / crossover)
- IPv4 addressing

## 🚀 Getting Started

1. Open `lab1.pkt` in Cisco Packet Tracer
2. Follow the instructions in the PDF/DOCX guide
3. Complete the exercises and verify all connectivity
